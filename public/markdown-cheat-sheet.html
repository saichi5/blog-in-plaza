１．見出し

冒頭に「#」を入力すると、見出しを作ることができます。「#」の数で見出しの大きさを変えられます。「#」と「見出し」の間に半角スペース1文字必要


[入力例]

# 見出し１

## 見出し２

### 見出し３

#### 見出し４


[表示結果]

見出し１

見出し２

見出し３

見出し４

２．段落

空白行を入れることで、段落となります。

空白行を入れる前

[入力例]

ドッグランのご利用は、マナーを守って

皆様が気持ちよく利用できるようご協力を．．．

[表示結果]

ドッグランのご利用は、マナーを守って皆様が気持ちよく利用できるようご協力を．．．


空白行を入れた後

[入力例]

ドッグランのご利用は、マナーを守って

(空白行)

皆様が気持ちよく利用できるようご協力を．．．

[表示結果]

ドッグランのご利用は、マナーを守って

皆様が気持ちよく利用できるようご協力を．．．

３．改行

文章の最後に半角スペースを２つ以上入力することで改行されます。

半角スペースを入れる前

[入力例]

ドッグランのご利用は、マナーを守って

皆様が気持ちよく利用できるようご協力を．．．

[表示結果]

ドッグランのご利用は、マナーを守って皆様が気持ちよく利用できるようご協力を．．．


半角スペースを入れた後

[入力例]

ドッグランのご利用は、マナーを守って␣␣

皆様が気持ちよく利用できるようご協力を．．．

[表示結果]

ドッグランのご利用は、マナーを守って

皆様が気持ちよく利用できるようご協力を．．．

４．文字の装飾(イタリック、太文字、取り消し線)

イタリック：
・文を「*」アスタリスク１個ずつで囲むと、イタリックになります。
太文字：
・文を「**」アスタリスク２個ずつで囲むと、太文字になります。
取り消し線：
・文を「~~」（波線）２つで囲むと、取り消し線が表示されます。

[入力例]

*アスタリスク１個でイタリック*

**アスタリスク２個で太文字**

~~波線２個で取り消し線~~

[表示結果]

アスタリスク１個でイタリック

アスタリスク２個で太文字

波線２個で取り消し線

５．リンク

[表示テキスト](URL)でリンクに変換されます。

[表示結果]

７．水平線

３つ以上、– (ハイフン)、_ (アンダースコア)、* (アスタリスク)を連続して記述している行は水平線（罫線）となります。

[入力例]

---  （ハイフン）

___  （アンダースコア）

***  （アスタリスク）

[表示結果]




８．リスト

箇条書きリスト：

文字の前に「*」アスタリスクをつけることで、箇条書きになります。「*」と「項目」の間に半角スペース1文字必要

[入力例]

* 項目

* 項目

* 項目

[表示結果]

  • 項目
  • 項目
  • 項目

番号付きリスト：

数字とピリオドを入力することで番号をふることができます。番号は自動的に採番されるため、すべての行を1.と記述するのがお勧めです。「1.」と「項目」の間に半角スペース1文字必要

[入力例]

1. 項目

1. 項目

1. 項目

[表示結果]

  1. 項目
  2. 項目
  3. 項目

９．文字色の変更

Markdown記法では、テキストカラーの変更は対応していないため、文字色を変えるためには、直接HTMLタグを記述する必要があります。

[入力例]

夕焼け空が<span style="color:red">赤い</span>のは、太陽が遠ざかり、波長の短い<span style="color:blue">青</span>が散乱するから

[表示結果]

夕焼け空が赤いのは、太陽が遠ざかり、波長の短い青が散乱するから


１０．中央揃え

文を中央に配置することができます。

[入力例]

<center>中央揃え</center>

[表示結果]

中央揃え


１１．コードの挿入

コードのような等幅文字列を挿入できます。

[入力例]

```TypeScript:import.ts

# コード

import type { Post } from '@/data';


export default function (params){

  const id = params.id;

  return (something);

}

```

[表示結果]

      
# コード
import type { Post } from '@/data';

export default function (params){
    const id = params.id;
    return (something);
}
